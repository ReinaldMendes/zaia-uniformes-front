┌─────────────────────────────────────────────────────────────────────────┐
│ 🔄 FLUXO DE DADOS                                                       │
└─────────────────────────────────────────────────────────────────────────┘

LOGIN:
   User → login.html → api.login() → Backend → JWT Token → localStorage

CARREGAR CONTEÚDO:
   Dashboard → admin.js → api.getContent() → Backend → Renderizar cards

CRIAR CONTEÚDO:
   Form → admin.js → api.createContent() → Backend → Atualizar lista

CARREGAR NA LANDING PAGE:
   index.html carrega → dynamic-content-loader.js executa
   → Busca elementos com data-content
   → Requisita /api/content
   → Preenche elementos com valores

ATUALIZAR LANDING PAGE:
   Usuario edita no CMS → Backend salva → Landing page recarrega
   → dynamic-content-loader executa → Novo conteúdo aparece

┌─────────────────────────────────────────────────────────────────────────┐
│ 📊 ENDPOINTS DA API                                                     │
└─────────────────────────────────────────────────────────────────────────┘

AUTH:
   POST   /api/auth/login               Login
   GET    /api/auth/me                  Dados do usuário

CONTEÚDO:
   GET    /api/content                  Listar todos
   GET    /api/content/:key             Obter por chave
   POST   /api/content                  Criar novo
   PUT    /api/content/:key             Editar
   DELETE /api/content/:key             Deletar
   POST   /api/content/upload/image     Upload de imagem

PARCEIROS:
   GET    /api/partners                 Listar
   POST   /api/partners                 Criar
   DELETE /api/partners/:id             Deletar

USUÁRIOS:
   GET    /api/users                    Listar
   POST   /api/users                    Criar
   PUT    /api/users/:id                Editar
   DELETE /api/users/:id                Deletar

📌 Todas as rotas (exceto login) requerem:
   Authorization: Bearer TOKEN_JWT
